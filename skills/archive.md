# Archive Completed Stories

Run `ralph-archive` to archive completed PRD stories to `docs.local/prd-archive/`.

## Instructions

1. If user specifies an app (e.g., `/archive frontend`), run:
   ```bash
   source ~/.config/ralph/ralph.zsh && ralph-archive <app>
   ```
2. Otherwise, run:
   ```bash
   source ~/.config/ralph/ralph.zsh && ralph-archive
   ```

Execute the command now.
